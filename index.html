设置与说明

使用说明：

1. 选择预设的乐队颜色或输入自定义颜色值

2. 可以切换是否显示歌词

3. 点击"开始应援"进入全屏应援模式

4. 在全屏模式下，点击右上角的"✕"按钮退出

预设颜色对应乐队：

• 红色 (#ee1166) - Vivid Bad Squad!

• 黄色 (#ff9900) - Wonderland X Showtime

• 绿色 (#88dd44) - MORE MORE JUMP!

• 蓝色 (#4455dd) - Leo/need

• 紫色 (#884499) - 25时，Nightcord见

• 青色 (#39c5bb) - VIRTUAL SINGER

提示： 自定义颜色值应为6位十六进制代码，以"#"开头。
